Самостоятельное задание 3.1 - Работа с текстом

Twitter Twitter Bootstrap

Twitter Bootstrap Свободный набор инструментов для создания сайтов и веб-приложений.

Включает в себя HTML и CSS шаблоны оформления для типографики, веб-форм, , меток, блоков навигации и прочих компонентов веб-интерфейсов, включая JavaScript расширения.

Сетки — заранее заданные размеры колонок, которые можно сразу же использовать, например ширина колонки 140px относится к классу span2, который можно сразу же использовать в CSS описании документа.

Типографика — Описания шрифтов, определение некоторых классов для шрифтов, таких как код, цитаты и т. п.

Медиа — Представляет некоторое оформление изображениями и видео.
